332 /*
333  Log may crash when the shared library loaded by deepbind and call this log API.
334  The reason is symbol copy-relocation, About copy-relocation, read here:
335  https://stackoverflow.com/questions/37296995/dynamic-loading-of-shared-library-with-rtld-deepbind
336  Setting the symbol to weak allows the deepbind library to use the first symbol without error.
337  If there is still a problem, you can use macros MBLOG_LIBRARY_DEEPBIND.
338  but this will cause a little slower when calling log API.
339 */
340 #ifdef MBLOG_LIBRARY_DEEPBIND

342 #else
